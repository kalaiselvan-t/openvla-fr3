## Key Features

- **VLA Stack** - Integration of FR3, ROS2 and MoveIt in Gazebo environment
- **Quantization** - 4-bit quantization of OpenVLA to run on consumer GPUs
- **Teleoperation** - MoveIt Servo based teleoperation node for training and testing
- **Docker** - Containerized development environment
